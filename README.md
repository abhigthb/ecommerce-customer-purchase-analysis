# E-Commerce Customer Purchase Analysis

## Project Title
E-Commerce Customer Purchase Analysis for Marketing Insights

## Objective
Analyze customer purchase data to identify top customers, popular products, and high-revenue cities.

## Tech Stack
- Oracle SQL
- Microsoft Excel (Pivot Tables)
- Python (Pandas + Matplotlib)

## Features
- Total purchase calculation
- Customer spending analysis
- Product popularity analysis
- City-wise revenue analysis
- Data visualizations (Bar & Pie charts)

## How to Run
1. Run `ecommerce_analysis.sql` in Oracle APEX
2. Open `Ecommerce_Analysis_Excel.xlsx`
3. Run `ecommerce_analysis.py` using Python
